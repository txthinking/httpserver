这是一个非常简单的 http static server, 通常用于测试

> **如果你需要一个正式的 HTTP2 web server, 试试 [nico](https://github.com/txthinking/nico)**

### 用 [nami](https://github.com/txthinking/nami) 安装

```
$ nami install github.com/txthinking/httpserver
```

### 使用

```
$ httpserver
```
